# FLEX-BOX #

## 基本属性 ##

### 1.flex container属性 ###


#### flex-direction ####
> * row(默认) 从左到右，行
> * row-reverse 从右到左，行
> * column 从左到右，列
> * column-reverse 从右到左，列

#### flex-wrap ####
> * nowrap(default) 强制一行
> * wrap 自动换行
> * wrap-reverse 自动换行，下一行的位置反转

#### flex-flow ####
> * flex-flow 将以上两种属性结合，如flex-flow: row wrap //即容器中的元素按水平方向排列，自动换行。

#### justify-content  ####
>用于分配container填充item后所剩余的空间
